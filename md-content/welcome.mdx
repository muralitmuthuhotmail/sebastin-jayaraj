## Quick Start

Get your documentation site running in under 2 minutes:

```bash
# Clone and start with Docker
git clone https://github.com/muralitmuthuhotmail/findoora-docs.git
cd findoora-docs
docker-compose up -d

# Visit http://localhost:3000
```

## Built With Modern Tech

- **Next.js 15** with App Router and React Server Components
- **TypeScript** for type safety and better developer experience
- **Tailwind CSS** for utility-first styling and theming
- **MDX** for Markdown with embedded React components
- **Docker** for easy deployment and development
